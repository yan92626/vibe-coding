> 一份从入门到精通的完整指南，帮助你掌握与 AI 协作编程的艺术

## 什么是 Vibe Coding？

Vibe Coding 是一种新兴的编程方式，核心理念是**凭感觉和直觉快速编码，通过与 AI 的高效协作实现快速迭代**，而不是过度规划和设计。它代表了从"工程学"到"创作"的范式转变，强调流动性、创造性和快速反馈循环。
